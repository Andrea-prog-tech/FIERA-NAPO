# Fiera Napoleonica – Pronto per Vercel

Questo progetto Next.js (app router) è già configurato con **Tailwind** e **Framer Motion**.

## Inserire l'immagine del cavallo
- Metti il tuo file dentro la cartella `public/` con nome **`napoleon-hero.png`** (sovrascrive il segnaposto).
- Oppure modifica `app/page.jsx` e cambia il valore di `src` nel tag dell'immagine.

## Deploy su Vercel (Upload ZIP)
1. Comprimi l'intera cartella del progetto (o usa lo ZIP allegato).
2. Vai su vercel.com → **Add New → Project → Upload**.
3. Carica lo ZIP → **Deploy**.

## Sviluppo locale (opzionale)
```bash
npm install
npm run dev
# apri http://localhost:3000
```
